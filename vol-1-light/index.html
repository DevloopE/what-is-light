光是什么?

—— 四十个角度的追问

围绕一个问题: 光到底是什么? 从 Fermat 的几何直线到真空零点场, 从蜜蜂眼里的紫外到银河中心黑洞旁的光锥, 四十个具体可计算的子问题展开一条由浅入深的阶梯 — 光线、波动、速度能量动量、量子热辐射、与物质的相互作用、信息载体、宇宙信号、生命感知. 每一层描述都是真的, 但没有一层是完整的. 每一节从一个能观察到的现象出发, 第一性原理推到一个具体的数字.

第 1 部分光线: 从费马到斯涅尔的几何学 把光当作一条理想射线就足够 — 但它在短波长极限下才严格成立
  1. 光为什么按直线传播?—— 费马原理与几何光学的波动根据
  2. 光为什么在水里"变慢"?—— 从电子响应推导折射率
  3. 彩虹为什么总挂在太阳对面 42°?—— 从斯涅尔到笛卡儿的几何极值
  4. 光为什么会弯曲 — 从折射到引力透镜?—— 折射与广义相对论的同一件事
  5. 阳光如何穿过一张玻璃? 多少能量返回?—— 菲涅耳方程、Brewster 角与全内反射
第 2 部分光波: 干涉、衍射、偏振 两束光能叠加出黑暗, 一束光能绕过障碍物 — 射线描述开始让位于波
  1. 两束光为什么能叠加出黑暗?—— 杨氏双缝与相位的代数
  2. 光为什么能绕过障碍物?—— 惠更斯原理与单缝衍射的 sinc 图案
  3. 衍射极限让望远镜"看不到"什么?—— 瑞利判据与角分辨率
  4. 光为什么会偏振?—— 电磁波的横波本质与 Brewster 角
  5. 激光为什么能保持方向不散?—— 相干性、受激辐射与束腰扩张
第 3 部分光的速度、能量与动量 光输运了什么? 速度从哪来, 能量和动量各自如何进入方程
  1. 光速究竟怎么被测出来?—— 从 Römer 到 Fizeau 到现代
  2. 光速为什么恰好是 299,792,458 米每秒?—— 麦克斯韦方程组自然地给出 c = 1/√(ε₀μ₀)
  3. "相速度"与"群速度"哪个是光的真速度?—— 信息、包络与"超光速"错觉
  4. 光竟然有动量?—— 辐射压、太阳帆与激光冷却
  5. 阳光每秒在地面送来多少能量?—— 从太阳参数反推太阳常数
第 4 部分光的量子性与热辐射 波动图景走到头 — 光必须是"量子", 而真空不空
  1. 光子从太阳核心走到表面要多久?—— 随机游走与光学深度
  2. 为什么炉火先红后白?—— 维恩位移、斯特藩玻尔兹曼与普朗克公式
  3. 光电效应为什么粉碎了经典波动图景?—— 爱因斯坦 1905 的光量子假说
  4. 光到底是波还是粒子?—— 单光子干涉与波粒互补
  5. 真空里真的没有光吗?—— 零点能、Casimir 效应与"虚"光子
第 5 部分光与物质的相互作用 从带隙到激光冷却 — 光遇到物质时发生什么
  1. 玻璃为什么透明, 金属为什么反射?—— 带隙与等离子体频率
  2. 肥皂泡为什么会有彩色?—— 薄膜干涉与反射相位
  3. CD 和 DVD 上的彩虹从哪来?—— 光栅方程与多缝干涉
  4. Compton 散射为什么是"光是粒子"的关键实验?—— 1923 年 X 光与电子的弹性碰撞
  5. 激光冷却怎么把原子冷到 nK?—— 用光的动量拦住原子
第 6 部分光与信息 当代通信、传感与密码的物理基础
  1. 光纤里的光怎么传 100 公里不散?—— 全内反射、低损耗硅玻璃与 EDFA
  2. 一根光纤能同时送多少路信号?—— WDM 与 Shannon 容量极限
  3. 激光雷达怎么测毫米级的距离?—— ToF、FMCW 与干涉测距
  4. 全息图为什么能存储 3D 信息?—— 记录波前的相位
  5. 量子密钥分发为什么"不可破解"?—— 不可克隆定理与 BB84 协议
第 7 部分光与宇宙 宇宙尺度上的信号 — 光如何携带宇宙学的全部信息
  1. 红移告诉我们宇宙在膨胀 — 怎么告诉的?—— Hubble 定律与宇宙学红移的起源
  2. 引力透镜如何称出看不见的暗物质?—— 用光的弯曲测量大质量的分布
  3. 中子星表面出发的光, 为什么会变红?—— 引力红移与广义相对论的钟
  4. CMB 各向异性的第一个声学峰藏着什么?—— 声学振荡、曲率与宇宙学参数
  5. 脉冲星的光为什么像钟摆一样精确?—— 中子星、磁偶极辐射与纳赫兹引力波
第 8 部分光与生命 看见、发光、感知 — 生物化学如何把光变成信号
  1. 光合作用的第一步用到了什么量子效应?—— 激子传输与可能的量子相干
  2. 人眼真能看到一个光子吗?—— 视杆细胞、视黄醛与神经放大
  3. 人眼看不见紫外, 蜜蜂却看得见 — 为什么?—— 视锥色素的吸收谱与"蜜导"
  4. 萤火虫的冷光是怎么来的?—— 生物发光的化学与量子效率
  5. 眼睛的分辨极限真的就是物理极限吗?—— 瞳孔、视锥密度与衍射的巧合优化
