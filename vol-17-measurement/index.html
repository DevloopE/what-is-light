测量是什么?

—— Measurement · 三十节

米、秒、千克、GPS、温度计、误差棒和宇宙距离共同说明: 测量是把世界接到标准上。 本卷按六个部分展开: SI 标准 → 时间和频率 → 长度/质量/电学 → 传感器 → 误差和噪声 → GPS/宇宙尺度. 每一节都从一个具体问题出发, 用一两个方程和一个数量级估算把概念落到可观察世界。

第 1 部分标准从哪里来 单位、定义、可复现
  1. 为什么测量需要标准?—— 可追溯性
  2. 米为什么来自光速?—— 长度变成时间
  3. 秒为什么来自铯原子?—— 频率标准
  4. 千克为什么来自 Planck 常数?—— Kibble balance
  5. SI 单位为什么要固定常数?—— 人造物到自然常数
第 2 部分日常仪器 尺、秤、温度计、传感器
  1. 尺子为什么也需要校准?—— 热膨胀与刻度
  2. 体重秤量的是质量还是重量?—— 力传感器
  3. 温度计到底在测什么?—— 热平衡
  4. 压力表为什么能读出气体状态?—— 弹性形变
  5. 相机测光为什么会被骗?—— 动态范围
第 3 部分误差和噪声 误差棒、统计、系统偏差
  1. 误差棒为什么不是装饰?—— 不确定度
  2. 多测几次为什么能变准?—— $1/\sqrt{N}$
  3. 系统误差为什么不会被平均掉?—— 校准偏差
  4. 信号和噪声怎样分开?—— SNR
  5. 分辨率和精度为什么不同?—— 仪器最小刻度
第 4 部分空间和导航 三角测量、GPS、惯导
  1. 三角测量为什么能量远处?—— 视差几何
  2. GPS 为什么需要四颗卫星?—— 时间偏差也要解
  3. GPS 为什么必须修正相对论?—— 钟速差
  4. 手机怎样知道自己在转?—— 加速度计和陀螺
  5. 激光雷达怎样把时间变成距离?—— 飞行时间
第 5 部分微观测量 原子、质子、量子极限
  1. 原子大小怎么知道?—— 散射与光谱
  2. 质子半径怎么量?—— 电子/缪子氢与散射
  3. 电子电荷为什么能单个测?—— Millikan 油滴
  4. 测量会不会扰动被测物?—— 量子极限
  5. 极弱信号怎样被锁相放大?—— 相干积分
第 6 部分宇宙尺度 距离阶梯、红移、模型
  1. 宇宙距离怎么知道?—— 距离阶梯
  2. 恒星视差为什么这么难?—— 角度极小
  3. 标准烛光为什么能当尺?—— 亮度距离
  4. 红移为什么能估距离?—— Hubble-Lemaitre 定律
  5. 测量是不是发现一个数?—— 模型、标准与世界
