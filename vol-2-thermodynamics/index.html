第二卷

热力学与统计力学

—— 三十节

经典热力学的几个定律和统计力学的几把工具, 已经被讲烂了. 但每个人都讲, 不代表都讲清楚了. 这一卷的三十节, 一半在重做基础 (熵到底是什么、配分函数 $\log Z$ 究竟为什么是热力学的全部、Legendre 变换怎么把内能 / 焓 / 自由能串起来), 一半在用这些工具往前走 — 相变、量子统计、非平衡涨落、信息熵到黑洞熵. 写法和第一卷一致: 从一个具体问题出发, 第一性原理推到一个具体的数字; 但**每节都必须把一个被滥用的 cliché 概念严格地讲一遍**.

第 1 部分经典热力学的根 从微观计数到 Carnot 上限 — 把"熵"和"自由能"重新定义清楚
  1. 第二定律到底是怎么"自然"出现的?—— 从 Boltzmann 的 $S = k\ln\Omega$ 推起
  2. 内能、焓、自由能, 凭什么是它们?—— Legendre 变换的几何与对偶
  3. 化学势 $\mu$ 究竟刻画什么?—— 粒子愿不愿意来的能量价
  4. 第三定律 — 为什么 $T\to 0$ 时熵走向常数?—— Nernst 假设与残余熵
  5. Carnot 效率 $1-T_c/T_h$ 凭什么是上限?—— 可逆引擎与热力学的"不公平"
第 2 部分统计力学的核心机器 配分函数、系综、Boltzmann 因子 — 把"微观"翻译成"宏观"
  1. 配分函数 $Z$ — 为什么 $\log Z$ 就是热力学的全部?—— 一个生成函数撑起所有方程
  2. 微正则、正则、巨正则该怎么挑?—— 三种系综的等价性证明
  3. $e^{-\beta E}$ 凭什么是它?—— 从最大熵原理推出 Boltzmann 分布
  4. 理想气体的所有性质, 都能从 $Z$ 完整推出吗?—— 一个例子从头做到底
  5. 涨落为什么按 $1/\sqrt N$ 缩小?—— 中心极限定理与热力学极限
第 3 部分相变与临界现象 液-气、铁磁、超流 — 不同物质, 同样的临界指数
  1. van der Waals 气体的"凝结"凭什么是 Maxwell 等面积律?—— 一阶相变与共存线
  2. 二阶相变与对称破缺 — Landau 平均场究竟做了什么?—— 序参量与有效势
  3. 临界指数为什么不依赖材料?—— Universality 类的物理基础
  4. 重整化群 — 怎么把"尺度不变"做成数学工具?—— Kadanoff-Wilson 的逐级粗粒化
  5. Ising 模型: 一维精确, 二维 Onsager, 三维 RG—— 三种维度三种命运
第 4 部分量子统计 玻色 vs 费米 — 同一框架, 两种命运
  1. 玻色子和费米子的化学势, 为什么命运完全相反?—— $\pm 1$ 在分布函数里的全部影响
  2. Bose-Einstein 凝聚 — 化学势贴上零的那一刻—— 宏观量子态的诞生
  3. 费米气体的简并压怎么撑起白矮星?—— Chandrasekhar 极限的推导
  4. 黑体辐射在统计力学里的完整推导—— 光子气体的 $Z$ 算法
  5. 声子: 把固体振动变成玻色气体—— Debye 与 Einstein 模型
第 5 部分非平衡热力学 从 Boltzmann 方程到涨落耗散 — 时间之箭怎么进物理
  1. Boltzmann 方程 — Newton 微观如何变成宏观流体?—— 动理论的奠基
  2. H 定理与 Loschmidt 反对 — 时间反演和不可逆怎么共存?—— Boltzmann 与他的批评者
  3. 涨落耗散定理 — 涨落怎么决定耗散?—— Kubo 关系与线性响应
  4. Onsager 倒易关系 — 线性响应的隐藏对称—— 时间反演下的等式
  5. Arrhenius 定律为什么是指数形?—— 反应速率从势垒得到
第 6 部分信息热力学与黑洞 从 Maxwell 妖到 Bekenstein 熵 — 信息也是物理量
  1. Maxwell 妖与 Landauer 原理 — 删 1 比特要花 $k_BT\ln 2$ 焦耳?—— 信息有热力学代价
  2. Jarzynski 等式 — 非平衡过程也能算自由能?—— 一个奇怪的恒等式
  3. Crooks 涨落定理 — 第二定律的微观陈述—— 时间反演下的概率比
  4. Bekenstein-Hawking 熵为什么是面积而不是体积?—— 视界面积 / 4 的物理
  5. 全息原理 — 体积里的信息真的存在边界面上吗?—— 't Hooft、Susskind、AdS/CFT
