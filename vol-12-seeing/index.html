第十二卷

看见是什么?

—— Seeing · 三十节

眼睛、显微镜、望远镜、全息图和深度知觉共同回答: 光怎样变成可用的信息。 本卷按六个部分展开: 眼球光学 → 视网膜与知觉 → 深度 → 显微镜极限 → 望远镜 → 成像与全息. 每一节都从一个具体问题出发, 用一两个方程和一个数量级估算把概念落到可观察世界。

第 1 部分眼睛作为光学仪器 焦距、调节、瞳孔、像差
  1. 眼睛是不是一台相机?—— 透镜、光圈与视网膜
  2. 眼睛怎样从远处切到近处?—— 调节与屈光度
  3. 近视为什么会把远处变糊?—— 焦平面错位
  4. 为什么边缘的光更难聚焦?—— 球差、色差与散光
  5. 瞳孔为什么不能无限缩小?—— 景深与衍射的折中
第 2 部分视网膜和大脑 光子、感受器、适应、恒常
  1. 视杆和视锥怎样数光子?—— 视紫红质与放大级联
  2. 视觉暂留是真的吗?—— 时间积分与闪烁融合
  3. 大脑怎样在不同光照下认出颜色?—— 白平衡的生物版
  4. 为什么边缘比均匀区域更醒目?—— 侧抑制
  5. 盲点为什么平时看不见?—— 填补与预测
第 3 部分深度与运动 双眼、视差、焦点、光流
  1. 双眼怎样给出深度?—— 立体视差
  2. 移动头部为什么能看出远近?—— 运动视差
  3. 散焦为什么也是深度线索?—— 点扩散函数
  4. 开车时世界为什么从中心向外流?—— 光流场
  5. 错觉为什么不是视觉失败?—— 先验与推断
第 4 部分显微镜 放大、NA、荧光、超分辨
  1. 显微镜为什么不只是把东西放大?—— 分辨率与数值孔径
  2. 显微镜为什么有分辨率极限?—— Abbe 衍射极限
  3. 透明细胞为什么也能看见?—— 相位变成强度
  4. 荧光显微镜为什么能标记分子?—— 激发与发射
  5. 超分辨为什么能绕过传统极限?—— 稀疏和非线性
第 5 部分望远镜 口径、收光、视宁度、干涉
  1. 望远镜为什么能看见更暗的星?—— 口径与光子数
  2. 望远镜分辨率为什么由口径决定?—— Airy 斑
  3. 大气抖动怎样被实时改正?—— 自适应光学
  4. 多台望远镜为什么能合成更大口径?—— 相干基线
  5. 太空望远镜为什么不只是更靠近星星?—— 去掉大气
第 6 部分成像新语言 相机、CT、全息、计算成像
  1. 数码相机怎样把光变成像素?—— 光电二极管与噪声
  2. 全息图为什么能记录三维?—— 干涉记录相位
  3. CT 为什么能从投影重建内部?—— Radon 变换
  4. 模糊照片为什么有时能复原?—— 点扩散函数反演
  5. 看见究竟是接收还是推断?—— 信息、噪声与先验
