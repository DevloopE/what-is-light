录音为什么需要采样率?

—— Nyquist 极限
CD 1982 Sony-Philips Red Book 选 44.1 kHz 是本节入口. 关键数字是 $f_s \ge 2 f_{\max}$ Nyquist; 22 kHz 听阈上限 + 余量; 44.1 = $4\times 11025$ 兼容 NTSC PCM 录像; 历史锚点是 Harry Nyquist 1928 Bell Labs, Claude Shannon 1949; 核心机制是 sampling theorem: 带限信号可由 $\ge 2B$ 采样完整重建. 下面从可见装置、最小方程、数量级、边界和听觉接收五层把它拆开.

CD 1982 Sony-Philips Red Book 选 44.1 kHz. CD 把连续声压变成每秒四万四千一百个数字, 这不是随便取的整数. 它要覆盖人耳上限, 还要适配早期 PCM 录像设备的工程格式.

Nyquist 和 Shannon 的结论很尖锐: 带限信号只要采样足够快, 可以完整重建. 反过来, 未带限就会 alias, 高频伪装成低频. 本章题目问的是 录音为什么需要采样率?, 答案不能只停在声音很响、很高或很远, 而要说明扰动怎样在介质里传播, 怎样被边界改写, 又怎样被耳朵或设备记录.

先固定量级: $f_s \ge 2 f_{\max}$ Nyquist; 22 kHz 听阈上限 + 余量; 44.1 = $4\times 11025$ 兼容 NTSC PCM 录像. 再固定历史锚点: Harry Nyquist 1928 Bell Labs, Claude Shannon 1949. 最后固定机制: sampling theorem: 带限信号可由 $\ge 2B$ 采样完整重建. 三件事放在一起, 才能避免把声学讲成一串孤立术语.

一、从现场到变量

sinc 重建公式看起来像数学理想, 工程上则由抗混叠滤波器、过采样和数字滤波近似完成. 声学变量通常成对出现: 压强对应体积压缩, 质点速度对应介质运动, 频率对应时间重复, 波长对应空间重复. 在 CD 1982 Sony-Philips Red Book 选 44.1 kHz 里, 这些变量不是同时显眼, 需要用图和仪器把它们拆出来.

从 CD 1982 Sony-Philips Red Book 选 44.1 kHz 入手, 先不要急着解释好不好听. 先问扰动在哪里变成压力, 在哪里变成速度, 在哪里被反射, 又在哪里被记录. 这些位置清楚后, 录音为什么需要采样率? 才有物理答案.

二、方程给出骨架

对 录音为什么需要采样率? 来说, 最小模型可以先写成下面的关系. 它故意忽略许多细节, 但保留了决定量级的核心变量:

$$ f_s\ge 2B. \tag{1} $$

式 (1) 的价值在于可换算. 只要知道介质参数、几何尺寸或频率, 就能估计一个可测结果. Harry Nyquist 1928 Bell Labs, Claude Shannon 1949 的意义也在这里: 历史上真正留下来的声学概念, 往往都是能把听感变成读数的概念.

采样率不等于音质的全部. 位深给动态范围, 时钟抖动影响相位, 录音链路的麦克风、房间和混音常比 44.1 与 48 kHz 的差别更显著. 这一层还没有处理所有真实复杂性, 但它能告诉我们哪个方向该变大, 哪个方向该变小, 以及哪些量根本不该混在一起比较.

录音为什么需要采样率? 的可听结果取决于接收端的时间窗和频带, 所以同一物理扰动在不同听者或设备上会留下不同痕迹.

三、把数字算回耳朵

现在把 $f_s \ge 2 f_{\max}$ Nyquist; 22 kHz 听阈上限 + 余量; 44.1 = $4\times 11025$ 兼容 NTSC PCM 录像 放回场景. 对声学来说, 量级常比精确末位更重要: 一毫秒的延迟可能被融合, 一百毫秒就可能成为回声; 二十微帕能触及阈值, 一百三十分贝会进入伤害范围.

$$ x(t)=\sum_{n=-\infty}^{\infty}x[n]\\,\mathrm{sinc}\!\left(\frac{t-nT_s}{T_s}\right). \tag{2} $$

式 (2) 把第二把尺放进来. 有时它是波长, 有时是能量, 有时是时间延迟或滤波权重. Nyquist 极限 的讲法要成立, 必须同时经得起这两把尺的检查.

把 CD 1982 Sony-Philips Red Book 选 44.1 kHz 放进实验室, 先估 $f_s \ge 2 f_{\max}$ Nyquist; 22 kHz 听阈上限 + 余量; 44.1 = $4\times 11025$ 兼容 NTSC PCM 录像 的数量级, 再决定麦克风、采样率、房间尺寸或材料厚度. 这一步决定了实验能不能看见想看的效应.

现场条件会把 录音为什么需要采样率? 从理想题变成测量题. 频响、反射、耦合、分层和动态范围若不说明, 同一公式会给出误导答案.

四、边界、仪器和听觉

真实声音一定会碰到边界. 墙面、水面、皮肤、乐器孔、耳道和麦克风振膜都不是被动背景, 它们会反射、透射、吸收或重新辐射能量. 连续信号 + 采样 + sinc interpolation 重建 应该把这些边界画成物理角色, 而不是只画声源本身.

声学实验常被环境偷走. 通风、桌面反射、人体遮挡、温度梯度都会混进 CD 1982 Sony-Philips Red Book 选 44.1 kHz 的信号, 因而 连续信号 + 采样 + sinc interpolation 重建 必须保留足够多的现场细节.

对 Nyquist 极限 来说, 接收端的限制不是麻烦, 而是现象的一半. 没有耳蜗和神经编码, 许多音色、响度和定位问题都只剩下裸波形.

五、适用边界和下一问

录音为什么需要采样率? 的简洁答案通常依赖线性、小振幅、稳定介质和清楚边界. 一旦声压过大、介质分层、声源运动、房间形状复杂或听觉任务改变, 同一个公式就要被修正或换成更细的模型.

把 sampling theorem: 带限信号可由 $\ge 2B$ 采样完整重建 和 $f_s \ge 2 f_{\max}$ Nyquist; 22 kHz 听阈上限 + 余量; 44.1 = $4\times 11025$ 兼容 NTSC PCM 录像 放在一起, 录音为什么需要采样率? 就不再靠形容词支撑. 它成为一条从声源到接收端的定量链.

$f_s \ge 2 f_{\max}$ Nyquist; 22 kHz 听阈上限 + 余量; 44.1 = $4\times 11025$ 兼容 NTSC PCM 录像 给了本章的量级. 量级的作用不是装饰, 它决定哪些效应能被听见、哪些只留在仪器里. 声学很容易被形容词带偏, 所以每个判断都要回到赫兹、帕斯卡、米每秒、毫秒或分贝.

Harry Nyquist 1928 Bell Labs, Claude Shannon 1949 这一行历史不是传记点缀. 它说明这个概念为什么会被发明: 有人需要把听感、乐器、通信、医学或海洋里的现象变成可重复测量的东西.
