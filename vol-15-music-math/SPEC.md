# SPEC — 第十五卷 · 音乐与数学 单篇文章规范

> 与 DevCollection 现有卷一致: 从一个可观察现象开始, 第一性原理, 落到一个具体公式、数量级或实验事实。本卷主角: Music and Math。

---

## 1 · 风格

1. 标题带问号或明确主题; 副标题用 `——` 给出关键概念
2. 摘要一段: 现象 → 机制 → 公式 → 数量级
3. 正文至少四个编号小节 (`一、` ...)
4. 至少两个编号公式, 至少一个具体数字代入
5. 至少一个历史/实验连接
6. 恰好一个图占位:

```html
<figure class="diagram-stub">
  <div class="stub-box">[此处应有一张图: 描述图想说的事]</div>
</figure>
```

## 2 · 本卷主线

泛音列 → 音程与调律 → Fourier 与音色 → 乐器耦合 → 节奏/群 → 压缩与信号处理

## 3 · 交付边界

本卷按第六、第七卷当前模式交付正文 HTML 与图占位, 不渲染 TikZ/SVG。若后续进入精修阶段, 再逐节把图占位替换为 `build/NN.svg`。
