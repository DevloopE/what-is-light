第三卷

粒子物理

—— 三十节

量子场论、规范对称、Feynman 图、Higgs 机制、重整化 — 每一个粒子物理的入门生都听过. 但听过和"真的会推" 之间隔了一道墙. 这一卷的三十节, 试图把那堵墙凿穿: 每一节从一个能观察到的现象 (一束 e⁺e⁻ 撞出 μ⁺μ⁻、一个自由中子的衰变、一个 Higgs 在 LHC 出现的 5σ 信号...) 出发, 第一性原理推到一个具体公式或数字, **每一节都必须把一个被频繁引用却很少真讲清楚的 cliché 概念严格地推一遍**.

第 1 部分QFT 的脚手架 从 Lagrangian 到路径积分 — 把"粒子" 重新定义为场
  1. 拉格朗日密度 $\mathcal L$ 凭什么能描述一个场?—— 从作用量原理开始
  2. 路径积分 $\int\!\mathcal D\phi\, e^{iS/\hbar}$ 是从哪儿来的?—— Feynman 1948 的"和所有路径"
  3. Feynman 传播子 — 自由场的全部内容—— 一个 Green 函数的几张面孔
  4. 二次量子化 — 把 $|\psi\rangle$ 升级成场算符—— 单粒子量子力学的局限
  5. 最小耦合 — 规范对称怎么"自动"给出相互作用?—— 从 $\partial\to D$ 的几何意义
第 2 部分对称性与守恒 Noether、规范、CPT — 把粒子物理变成群论的故事
  1. Noether 定理 — 连续对称就一定守恒?—— 一个被引用千万次的定理的真正证明
  2. U(1) 规范对称怎么给出电磁学?—— QED 的诞生
  3. SU(2) × U(1) — 弱电统一是怎么"接上"的?—— Glashow-Weinberg-Salam 的步骤
  4. SU(3) 与 QCD — 颜色对称的群论结构—— 三种"颜色"和八种胶子
  5. CPT 定理 — 为什么 C, P, T 单独可破缺, 三者必定守恒?—— Lorentz 不变性的副产物
第 3 部分标准模型与具体粒子 三代粒子谱 + Higgs + QCD + 弱相互作用 — 把它们组装起来
  1. 三代粒子谱 — 为什么是三代?—— 中微子振荡告诉我们的一件事
  2. Higgs 机制 — 自发对称破缺到底破的是哪个对称?—— 从 Mexican-hat 势开始
  3. 渐近自由 vs 夸克禁闭 — QCD 的两面—— 同一个耦合常数, 两种行为
  4. CP 破缺 — KAON 系统给我们看到了什么?—— Kobayashi-Maskawa 的预言
  5. 中微子振荡 — 它给中微子的小质量—— 从 Pontecorvo 到 SNO/Kamiokande
第 4 部分计算工具与发散 Feynman 图、重整化、跑动耦合、反常 — QFT 真正干活的部分
  1. $e^+e^-\to \mu^+\mu^-$ 的 Feynman 图怎么读?—— 一个具体例子的 leading-order 计算
  2. 重整化 — 发散是从哪里来, 怎么吸收掉的?—— 一个被神秘化的过程的真相
  3. 跑动耦合常数 — 为什么 $\alpha_s(M_Z) \ne \alpha_s(1\,\text{GeV})$?—— RG 流与β函数
  4. 反常 (anomaly) — 经典对称怎么在量子层面破缺?—— 三角图的奇迹
  5. Wick 转动 — Minkowski 到 Euclidean 路径积分怎么过去?—— 复平面里的一次旋转
第 5 部分实验与统计 加速器、探测器、截面、寿命、5σ — 怎么把理论变成数据
  1. 加速器的物理 — LHC 怎么造出 13 TeV 的对撞?—— 同步加速器与束流冷却
  2. 探测器 — ATLAS / CMS 怎么"看见"一个 Higgs?—— 从层层探测器到事件重建
  3. 截面 $\sigma$ — 实验和理论的桥梁—— 一个有面积单位的概率
  4. 衰变率 $\Gamma$ 与寿命的关系—— 自由中子能活多久?
  5. 5σ 显著性 — 凭什么是 5 而不是 3?—— look-elsewhere 与试验家数
第 6 部分BSM — 标准模型之外 暗物质、强 CP、baryogenesis、GUT、SUSY — 还没解决的事
  1. 暗物质粒子的候选 — WIMP, Axion, 还是更怪的东西?—— 不同候选者的截面与遗迹丰度
  2. 强 CP 问题 — 为什么 $\theta\approx 0$?—— Peccei-Quinn 与轴子
  3. 原初 baryogenesis — 物质 vs 反物质的不对称—— Sakharov 三条件
  4. 大统一 (GUT) — 把 SU(3)×SU(2)×U(1) 嵌入更大的群?—— SU(5)、SO(10) 与质子衰变
  5. 超对称 (SUSY) — 为什么 30 年没找到?—— 一个漂亮理论与一个 LHC 失望
