第五卷

引力是什么?

—— 三十节

牛顿说引力是一种力, 在远处瞬间作用. 但这与他自己讲过的因果与相对运动相抵触, 他自己也讲不清. Einstein 1915 年给出的答案是: 引力不是力 — 它是时空的几何, 物质告诉时空怎么弯, 弯的时空告诉物质怎么动. 本卷追问"引力到底是什么", 从 Einstein 的电梯思想实验出发, 一路推到 Schwarzschild 黑洞、引力波、Friedmann 宇宙、Hawking 辐射、量子引力的边界. 每一节落到一个能算出来的具体数字: GPS 一天偏 38 微秒、Sun 的史瓦西半径 3 公里、Mercury 进动 43"/世纪、GW150914 应变 10⁻²¹.

第 1 部分等效原理 — 引力为什么不是力 从 Einstein 的电梯思想实验, 到引力红移、光线偏折、Minkowski 时空
  1. 引力为什么不是一种力?—— Einstein 的电梯与等效原理
  2. 时间真的会变快慢吗?—— 引力红移与 Pound-Rebka
  3. 光为什么会被太阳偏折?—— 1919 日食与 1.75 角秒
  4. Minkowski 时空 — 距离怎么定义?—— 度规、光锥、世界线
  5. 匀加速观察者的世界长什么样?—— Rindler 坐标与视界
第 2 部分弯曲时空 — 度规、测地线、曲率 把"几何"翻成可算的张量: $g_{\mu\nu}$, $\Gamma^\lambda_{\mu\nu}$, $R^\rho{}_{\sigma\mu\nu}$
  1. 度规 $g_{\mu\nu}$ 究竟编码了什么?—— 线元、时空间隔、对称性
  2. 自由下落的轨迹怎么算?—— 测地线方程与 Christoffel 符号
  3. 为什么"平行"在弯曲时空里有歧义?—— 协变导数与平行运输
  4. 潮汐力是怎么从曲率张量里浮出来的?—— Riemann 张量与测地偏离
  5. 为什么 Einstein 张量是这个组合?—— Ricci, $R$, Bianchi 与守恒
第 3 部分Einstein 场方程 $G_{\mu\nu}=8\pi G\,T_{\mu\nu}/c^4$ 怎么来的, 它在弱场和强场里给出什么
  1. $G_{\mu\nu} = 8\pi G\,T_{\mu\nu}/c^4$ 是怎么写下来的?—— 物质告诉时空怎么弯
  2. Schwarzschild 解 — 球对称真空里时空长什么样?—— $r_s = 2GM/c^2$
  3. Newton 引力是怎么从 GR 里浮现的?—— 弱场极限与 $g_{00}\approx -(1+2\Phi/c^2)$
  4. 引力波 — 时空真的会涟漪吗?—— 线性化 GR 与 $h_{\mu\nu}$
  5. Einstein-Hilbert 作用量怎么变出场方程?—— $S = \int R\sqrt{-g}\,d^4x/16\pi G$
第 4 部分黑洞 视界、潮汐、Kerr 旋转、Hawking 辐射、面积熵
  1. 视界 — 进得去, 出不来, 究竟是什么意思?—— Schwarzschild 视界与 Penrose 图
  2. 落入黑洞 — 你看到的与外面看到的有何不同?—— 固有时 vs 坐标时
  3. 旋转黑洞为什么有"能层"?—— Kerr 度规与 Penrose 过程
  4. 黑洞会蒸发吗?—— Hawking 温度 $T_H = \hbar c^3/8\pi G M k_B$
  5. 为什么黑洞熵正比于面积?—— 黑洞热力学四定律
第 5 部分宇宙学 FLRW 宇宙、Friedmann 方程、暗能量、CMB 与 BBN
  1. 宇宙真的是均匀各向同性的吗?—— 宇宙学原理与 FLRW 度规
  2. 膨胀宇宙的方程是什么样的?—— Friedmann 方程
  3. 为什么宇宙在加速膨胀?—— 宇宙学常数 $\Lambda$ 与暗能量
  4. CMB 是宇宙最早的快照吗?—— 2.725 K, $z\sim 1100$
  5. 为什么宇宙里 25% 是氦?—— 大爆炸核合成
第 6 部分检验与前沿 Mercury 进动、GPS、LIGO、EHT, 以及量子引力的边界
  1. Mercury 近日点为什么每百年多偏 43 角秒?—— GR 的第一个检验
  2. GPS 一天偏 38 微秒, 你怎么校正?—— SR 与 GR 在身边
  3. LIGO 怎么测出 $10^{-21}$ 的应变?—— 双黑洞合并 GW150914
  4. EHT 拍到的"黑洞照片"是什么?—— M87* 与 Sgr A* 的阴影
  5. 量子引力 — 为什么至今没解决?—— Planck 尺度与全息原理
